Downloading Data

We make sure you can access your hirepool data at any time to take it with you.

Find the download options in your user profile by clicking the small person icon from the hirepool menu. Once in your user profile there is a button labeled "Export My Data".

You'll find options there to export all the data you've entered into hirepool:

  • Opportunities: The job opportunities you enter into hirepool.
  • Events: The interviews you enter for each opportunity.
  • Interactions: The sections of interviews if you track multi-part interviews (like an on-site interview where you have a whiteboarding session, a behavioral interview, and lunch with the team).
  • Collaborations: The feedback shared by friends from your network if you used the Collaborators feature for an opportunity.
  • Interviewers: Details about the interviewers you track for Events and Interactions.
  • Resumes: Any resumes you've uploaded to hirepool.
